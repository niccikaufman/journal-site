## Run Locally

Clone the project

```bash
  git clone https://github.com/niccikaufman/journal-site.git
```

Go to the project directory

```bash
  cd journal-site
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
